# Southern Cross School Hub

## Installation
1. Download all files
2. Open `schoolhub.html` in any web browser
3. Register with role-specific codes:
   - Admin: SC2026ADMIN
   - Teacher: Any username
   - Parent: Any username

## Features
- Mobile responsive design
- Role-based dashboards
- Messaging system
- Student management
- Attendance tracking
- Fee management

## System Requirements
- Any modern web browser
- No internet connection required
- Data stored locally in browser
